Installation

  1. Copy all the files into your root directory . (You cant install it in sub directory , either install it on main domain or sub domain).
  2. Open browser and type [server_name]/directory_name.(e.g. localhost/kinventory)

Features

  • Built on Code Igniter Framework
    CodeIgniter is a powerful PHP framework with a very small footprint, built for developers who need a simple and elegant toolkit to create full-featured web applications.
  • Sales Management
    You Can sale your items and check Sales history of each invoice.
  • Purchase Management
    You can make a purchase and check Purchase History of each invoice.
  • Stock Management
    You can check your stock per each item / product you purchase or sale.
  • Product Management with Barcode
    You can add new items / products and either enter your own barcode or system will generate ot for you.
  • User Management
    Add users, assign role to each user.
  • Roles Permissions Management
    Add new roles and assign permissions to each role
  • Detailed Reporting
    You can check date wise sales and purchase reports.
  • Simple and Clean
    Its very easy and basic version, can easily be used anywhere.
